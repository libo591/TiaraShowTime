<h1>
  <span style="font-family:simhei;font-size:48px;">
    T-ara
  </span>
</h1>
<p>
</p>
<p>
</p>
<p>
  </p>
  <p>
    <span style="color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
      T-ara，拥有“韩国最百变女团” 韩国歌谣界“变色龙”等美誉，由
    </span>
      <strong style="font-weight:bold;">
        <span style="text-decoration:none;color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
          全宝蓝
        </span>
      </strong>
    <strong style="font-weight:bold;">
      <span style="color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
        、
      </span>
    </strong>
      <strong style="font-weight:bold;">
        <span style="text-decoration:none;color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
          李居丽
        </span>
      </strong>
    <strong style="font-weight:bold;">
      <span style="color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
        、
      </span>
    </strong>
      <strong style="font-weight:bold;">
        <span style="text-decoration:none;color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
          朴素妍
        </span>
      </strong>
    <strong style="font-weight:bold;">
      <span style="color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
        、
      </span>
    </strong>
      <strong style="font-weight:bold;">
        <span style="text-decoration:none;color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
          咸恩静
        </span>
      </strong>
    <strong style="font-weight:bold;">
      <span style="color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
        、
      </span>
    </strong>
      <strong style="font-weight:bold;">
        <span style="text-decoration:none;color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
          朴孝敏
        </span>
      </strong>
    <strong style="font-weight:bold;">
      <span style="color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
        、
      </span>
    </strong>
      <strong style="font-weight:bold;">
        <span style="text-decoration:none;color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
          朴智妍
        </span>
      </strong>
    <span style="color:#000000;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
      六名散发不同魅力的成员组成。组合名称T-ara源自英文Tiara，意指将成为歌谣界的女王。T-ara以无限挑战的团队宗旨完美诠释了百变的无尽吸引力及超高风格驾驭力，如今不仅在歌谣界发光发热，更是在影视界、艺能界小有建树，是韩国极具代表性的偶像团体之一。
    </span>
    <strong style="font-weight:bold;">
      <br/>
    </strong>
  </p>
  <div>
    <strong style="font-weight:bold;">
      <span style="font-size:20px;">
            
      </span>
    </strong>
  </div>

</div>
<h2>
  <div>
     
  </div>
  <div>
     
  </div>
    ˇT-araˇ组合简介
  <div>
  <span style="font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
    T-ara 造型百变，曲
  </span>
  <div style="text-align:center;margin-top:10px;margin-left:10px;float:right;font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
    <br/>
    <br/>
  </div>
  <span style="font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
    风百变，舞种百变的T-ara成员有着不同的个人魅力使其在各自的粉丝领域吸睛发亮。T-ara是
  </span>
	<span style="font-size:14px;">
	  M-net
	</span>
  <span style="font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
    公司准备了三年时间的女子组合，于2009年7月29日出道。由六位成员组成，她们分别是：童颜担当
  </span>
    <span style="font-size:14px;">
      全宝蓝
    </span>
  <span style="font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
    、门面担当
  </span>
    <span style="font-size:14px;">
      李居丽
    </span>
  <span style="font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
    、实力担当
  </span>
    <span style="font-size:14px;">
      朴素妍
    </span>
  <span style="font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
    、酷帅担当
  </span>
    <span style="font-size:14px;">
      含恩静
    </span>
  <span style="font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
    、性感担当
  </span>
    <span style="font-size:14px;">
      朴孝敏
    </span>
  <span style="font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
    、人气担当
  </span>
    <span style="font-size:14px;">
      朴智妍
    </span>
  <span style="font-family:arial, &#23435;&#20307;, sans-serif;font-size:14px;">
    。
  </span>
  <br/>
</div>
<div>
<p style="margin-top:0.4em;font-family:sans-serif;font-size:15px;">
  <span style="font-size:14px;color:#000000;">
    <br/>
  </span>
</p>
<p style="margin-top:0.4em;font-family:sans-serif;font-size:15px;">
  <span style="font-size:14px;color:#000000;">
    出道专辑
    <strong style="font-weight:bold;">
      《Absolute First Album》
    </strong>
    在2009年12月发行，并衍生几首热门单曲，包括谎言、TTL (Time to Love)、TTL Listen 2、Bo Peep Bo Peep和一如当初。跟着推出的迷你专辑
    <strong style="font-weight:bold;">
      《Temptastic》
    </strong>
    (2010)收录了热门单曲为什么这样及Yayaya。专辑
    <strong style="font-weight:bold;">
      《John Travolta Wannabe》
    </strong>
    (2011)中的Roly-Poly在Gaon年度排行榜中排名首位，并夺得多个奖项及提名。6个月后发行的迷你专辑
    <strong style="font-weight:bold;">
      《Black Eyes》
    </strong>
    (2011)产生出3首第一单曲：Cry Cry、我们不是相爱过吗和Lovey-Dovey，分别在
  </span>
    <span style="font-size:14px;color:#000000;">
      告示牌韩国K-pop Hot 100
    </span>
  <span style="font-size:14px;color:#000000;">
    排名第一。
  </span>
</p>
<p style="margin-top:0.4em;font-family:sans-serif;font-size:15px;"/>
<p>
  <span style="font-size:14px;font-family:&#23435;&#20307;, simsun;">
    <br/>
  </span>
</p>
<p>
  <span style="font-size:14px;font-family:&#23435;&#20307;, simsun;">
    此外《因你而疯》MV登上了韩国国内MV排行榜上首位。该MV在25日下午1点通过GOM TV正式公开，在
    <strong style="font-weight:bold;">
      不到两个小时的时间内就在该网站设立的MV排行榜上获得了冠军
    </strong>
  </span>
</p>
<p>
  <span>
    <br/>
  </span>
</p>
<p>
  <span style="font-size:14px;font-family:&#23435;&#20307;, simsun;">
    <strong style="font-weight:bold;">
      2011-12-23午夜由T-ara和Davichi成员共同演唱的《我们相爱不是吗》一经公开、开始掀起粉丝们暴发性反应位居榜首,席卷了歌谣界
    </strong>
    。 
    <strong style="font-weight:bold;">
      melon、Cyworld Music 、mnet、soribada、bugs等主要在线音乐网站中,组合t - ara和davichi的合唱曲《我们相爱不是吗》占据第1位的同时,tara的“cry cry”也都排在前五位,气势汹汹。
    </strong>
  </span>
</p>
<p>
  <strong style="font-weight:bold;">
    <span>
      <br/>
    </span>
  </strong>
</p>
<p>
  <strong style="font-weight:bold;">
    <span style="font-size:14px;font-family:&#23435;&#20307;, simsun;">
      此外T-ara的歌曲《Lovey Dovey》除登顶一月榜外 连续七周（一月一周~二月三周）
    </span>
  </strong>
</p>
<p>
  <span>
    <br/>
  </span>
</p>
<p>
  <span style="font-size:14px;font-family:&#23435;&#20307;, simsun;">
    时间位于榜单的TOP10，获得持久的人气。
    <strong style="font-weight:bold;">
      占据一季度全体的一位
    </strong>
  </span>
</p>
<p style="margin-top:0.4em;font-family:sans-serif;font-size:15px;">
  <span style="font-size:14px;color:#000000;">
    <br/>
  </span>
</p>
<p style="margin-top:0.4em;font-family:sans-serif;font-size:15px;">
  <span style="font-size:14px;color:#000000;">
    在2011年，T-ara以4.3亿与日本J-ROCK签约，准备在日本出道，是所有女子组合中
    <strong style="font-weight:bold;">
      最高价
    </strong>
  </span>
  <span style="font-size:14px;color:#000000;">
    。唱片公司确定为“
  </span>
    <strong style="font-weight:bold;">
      <span style="font-size:14px;color:#000000;">
        EMI音乐 (日本
      </span>
    </strong>
    <span style="font-size:14px;color:#000000;">
      )
    </span>
  <span style="font-size:14px;color:#000000;">
    ”
  </span>
  <span style="font-size:14px;color:#000000;">
    。第一首单曲Bo Peep Bo Peep(2009)在Oricon每周排行榜中排名第一，并售出49712个拷贝
  </span>
  <span style="font-size:14px;color:#000000;">
    ，是史上首队初出道而得榜首的韩国女子组合
  </span>
  <span style="font-size:14px;color:#000000;">
    。在2012年T-ara发行日本专辑《Jewelry Box》，高峰时在Oricon排行榜中排第2。当年7月发布的迷你专辑《
  </span>
    <span style="font-size:14px;color:#000000;">
      Day by Day
    </span>
  <span style="font-size:14px;color:#000000;">
    》(2012)在Gaon排行榜中排行第5，主题曲Day by Day顶峰时排行第2。2个月后发行迷你专辑《Mirage》(2012)，单曲Sexy Love顶峰时分别在Gaon排行榜和告示牌韩国K-pop Hot 100中排行第4位及第3位。
  </span>
</p>
<p style="margin-top:0.4em;font-family:sans-serif;font-size:15px;">
  <span style="font-size:14px;color:#000000;">
    2013年，T-ara于3月发布第6首日语单曲〈
  </span>
    <span style="font-size:14px;color:#000000;">
      Bunny Style!
    </span>
  <span style="font-size:14px;color:#000000;">
    〉
  </span>
  <span style="font-size:14px;color:#000000;">
    ，占据了Oricon每日及每周CD单曲排行榜的第二位。4个月后发布第7首日语单曲〈
  </span>
    <span style="font-size:14px;color:#000000;">
      Target
    </span>
  <span style="font-size:14px;color:#000000;">
    〉
  </span>
  <span style="font-size:14px;color:#000000;">
    8月，T-ara发行第2张日本专辑《Treasure Box》，占据了Oricon每日排行榜的榜首
  </span>
  <span style="font-size:14px;color:#000000;">
    。T-ara又成立了两个子团——
  </span>
    <span style="font-size:14px;color:#000000;">
      T-ara N4
    </span>
  <span style="font-size:14px;color:#000000;">
    及
  </span>
    <span style="font-size:14px;color:#000000;">
      QBS
    </span>
  <span style="font-size:14px;color:#000000;">
    。
  </span>
</p>
<p style="margin-top:0.4em;font-family:sans-serif;font-size:15px;">
  <span style="font-size:14px;color:#000000;">
    出道后加入过3个成员：
  </span>
    <span style="font-size:14px;color:#000000;">
      和荣
    </span>
  <span style="font-size:14px;color:#000000;">
    (2010)、
  </span>
    <span style="font-size:14px;color:#000000;">
      雅琳
    </span>
  <span style="font-size:14px;color:#000000;">
    (2012)及Dani(2013年8月)，后来
  </span>
    <span style="font-size:14px;color:#000000;">
      和荣
    </span>
  <span style="font-size:14px;color:#000000;">
    和
  </span>
    <span style="font-size:14px;color:#000000;">
      雅琳
    </span>
  <span style="font-size:14px;color:#000000;">
    分别在2012年7月及2013年7月退出
  </span>
  <span style="font-size:14px;color:#000000;">
    。T-ara每年会改变队长
  </span>
  <span style="font-size:14px;color:#000000;">
    <strong style="font-weight:bold;">
      现今队长为
    </strong>
  </span>
    <strong style="font-weight:bold;">
      <span style="font-size:14px;color:#000000;">
        居丽
      </span>
    </strong>
  <strong style="font-weight:bold;">
    <span style="font-size:14px;color:#000000;">
      。
    </span>
  </strong>
</p>
<p style="margin-top:0.4em;font-family:sans-serif;font-size:15px;">
  <span style="font-size:14px;color:#000000;">

    <strong style="font-weight:bold;">
      <span style="font-size:14px;color:#000000;">
        “Queen's”
      </span>
    </strong>
    ，意思是“女王所有的”
  </span>
  <span style="font-size:14px;color:#000000;">
